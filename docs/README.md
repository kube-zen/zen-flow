# Helm Chart Repository

This directory contains the Helm chart repository for zen-flow.

## Repository URL

```
https://kube-zen.github.io/zen-flow/charts
```

## Usage

```bash
# Add the repository
helm repo add zen-flow https://kube-zen.github.io/zen-flow/charts
helm repo update

# Install zen-flow
helm install zen-flow zen-flow/zen-flow \
  --namespace zen-flow-system \
  --create-namespace
```

## Chart Packages

Chart packages (`.tgz` files) and the repository index (`index.yaml`) are automatically generated by GitHub Actions when charts are updated.
